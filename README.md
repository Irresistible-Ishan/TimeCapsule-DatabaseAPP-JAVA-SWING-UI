# features :
* **Burn**: Permanently destroys the file from the local SSD or HDD by overwriting the storage sectors with randomized binary data, making recovery virtually impossible, even with advanced forensic tools.

* **Eliminate**: Completely removes the file from all connected sources, including linked cloud storage platforms and synchronized backups.

* **Save**: Stores the file locally in a predefined directory or user-configured storage location.

* **Export**: Transfers or synchronizes files to connected cloud storage services or linked external platforms.

* **Import**: Supports importing data from virtually any existing file format. This includes text documents, PDFs, images, and other media types. For image-based content, the system automatically performs OCR (Optical Character Recognition) and directly extracts and integrates the detected text.
